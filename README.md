## top 명령어

`top` 명령어는 시스템의 실시간 프로세스 정보를 표시하는 명령어입니다.

### 주요 옵션
| 옵션 | 설명 |
| ---- | ---- |
| `-d` | 업데이트 주기 설정 (초) |
| `-p` | 특정 PID를 모니터링 |

### 사용 예시
```sh
top -d 1
